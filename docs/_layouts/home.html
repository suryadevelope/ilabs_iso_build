vmConsole is a Linux virtual machine application for Android OS. It is built on top of QEMU system emulator and runs Alpine Linux â€” a small, yet powerful Linux distribution. This application lets you to do everything, like from playing with Bash scripts to performing a penetration test of remote server.

Powerful. Alpine Linux distribution has over 10000 packages in its repositories covering the majority of typical tasks. GCC, Python, Docker, Git, NMap, ViM and many other well-known tools are available there. We think that application will be useful for system administrators or people who just exploring the Linux world.

Secure. What happens inside VM is remaining to be inside VM. You can't mess up or brick your device by running something inside vmConsole. Note that you're given full root control over virtual machine without the need of rooting or installing customized version of Android.

Minimalistic. vmConsole design is minimal and provides only necessary UI elements and features in order to provide a stable and consistent software. It is not overloaded by garland of semi-working functions unlike certain similar applications.
